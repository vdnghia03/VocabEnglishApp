# 1.1 Tổng quan dự án (Project Introduction)

## 1. Thông tin chung
- **Tên dự án:** EnglishVocabApp
- **Mô tả ngắn:** Ứng dụng web hỗ trợ học từ vựng tiếng Anh nâng cao theo phương pháp lặp lại.
- **Ngày khởi tạo:** 19/12/2025

## 2. Mục tiêu (Goals)
- **Học tập:** Giúp người dùng ghi nhớ 20 từ vựng mỗi bài thông qua danh sách trực quan.
- **Dữ liệu:** Xây dựng kho dữ liệu sạch, chuẩn hóa (Clean & Standardized Data) để có thể tái sử dụng cho nhiều tính năng khác nhau (Flashcard, Quiz, Game...).
- **Trải nghiệm:** Giao diện đơn giản, tập trung vào nội dung, không gây xao nhãng.

## 3. Đối tượng sử dụng (Target Audience)
- Người học tiếng Anh 
- Người muốn ôn từ vựng

## 4. Công nghệ (Tech Stack)
- **Frontend:** REACT FRONTEND
- **Data:** JavaScript Objects (Export const) / JSON format.
- **Documentation:** Markdown.
