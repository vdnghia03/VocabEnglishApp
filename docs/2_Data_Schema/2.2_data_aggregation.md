# 2.2 Tập hợp Dữ liệu (Data Aggregation)

## 1. Mục đích
Thay vì import từng file bài học riêng lẻ vào ứng dụng chính (gây rối code), chúng ta sẽ sử dụng một file trung gian (Index) để tập hợp tất cả bài học thành một mảng dữ liệu duy nhất.

## 2. Cấu trúc File Index
File tập hợp thường được đặt tên là `index.js` nằm trong thư mục chứa dữ liệu (ví dụ: `data/index.js`).

### Quy trình:
1. **Import** từng object bài học từ các file lẻ.
2. **Gom** chúng vào một mảng lớn.
3. **Export** mảng đó ra ngoài để App sử dụng.

## 3. Mẫu Code (Code Snippet)
Ví dụ nội dung file `data/index.js`:

```javascript
// 1. Import các bài học
import { lesson1 } from './lesson1.js';
import { lesson2 } from './lesson2.js';
import { lesson4 } from './lesson4.js';
import { lesson10 } from './lesson10.js';

// 2. Tạo mảng chứa tất cả bài học
// Thứ tự trong mảng này sẽ quyết định thứ tự hiển thị trên Menu
export const allLessons = [
  lesson1,
  lesson2,
  lesson4,
  lesson10
];

/* Lợi ích:
- Khi cần lấy danh sách bài học để tạo Menu: dùng allLessons.map(...)
- Khi cần tìm bài học theo ID: dùng allLessons.find(...)
*/
```

## 4. Mở rộng (Scalability)

Khi có bài học mới (ví dụ lesson5.js):

Tạo file lesson5.js đúng chuẩn Schema (xem mục 2.1).

Mở file index.js.

Thêm dòng import và thêm vào mảng `allLessons`.
